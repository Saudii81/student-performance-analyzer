# Student Performance Analyzer

A simple Python project that analyzes student scores and provides performance insights.

## Features

- Calculate average, min, max scores
- Identify best student(s)
- Compute pass rate
- Assign grades (A–F)
- CLI output
- JSON input support
- Unit tests included
